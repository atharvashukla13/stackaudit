## Why This Copy Works

### Headline Analysis
- **"Your AI tools"** — personal, specific (not "AI tools in general")
- **"are costing too much"** — states the problem directly, creates urgency
- No jargon, no buzzwords, no "leverage" or "optimize" — just plain English

### Subheadline Analysis
- **"60 seconds"** — time commitment anchor reduces friction
- **"See exactly where"** — promises specificity, not vague advice
- **"your team overspends"** — frames it as a team decision, appeals to managers
- **Tool names listed** — signals we support the tools they actually use

### CTA Analysis
- **"My"** — possessive language increases ownership
- **"→"** — visual cue that this leads somewhere
- Not "Get Started" or "Sign Up" — no commitment language

## Objection Handling (In-Page)

| Objection | How We Handle It |
|-----------|-----------------|
| "Is this a sales pitch?" | Value shown BEFORE email ask. Honest "you're optimal" result. |
| "How do I know the numbers are right?" | PRICING_DATA.md linked, every number traces to official vendor page |
| "What if my stack is already good?" | "You're spending well" result with notify-me option |
| "Do I need to create an account?" | "No login required" in trust line |
| "Is my data safe?" | No PII stored, shared audits strip company info |

## Results Page Copy

### Savings Found
- **Hero:** "$X/mo" in large gradient text with pulse animation
- **Subtext:** "$Y/year" to amplify impact
- **Context:** "Current spend: $Z/mo across N tools"

### No Savings
- **Hero:** "✓ Optimized" in green
- **Message:** "Your AI stack is already well-optimized."
- **CTA:** "Get notified when new optimization opportunities apply to your stack."

### Per-Tool Recommendation
- **Format:** Tool name + current plan + savings amount
- **Action badges:** "Downgrade plan" / "Switch tool" / "Consolidate" / "Use Credex credits"
- **Confidence badges:** "High confidence" (green) / "Medium" (yellow) / "Low" (gray)
- **Reasoning:** 1-2 sentences of finance-literate explanation
